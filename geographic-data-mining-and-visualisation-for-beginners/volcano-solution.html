Solution to a volcano exercise

Try googling <volcano OSM Wiki>. Results should include a page describing how volcanoes are mapped in the OSM as one of results.

This page has a description that describes that natural=volcano is used to tag volcanoes.

So we want to hunt for volcanoes.

  1. Go to http://overpass-turbo.eu/s/zNU (this link places you at a specific area).
  2. Press the "Wizard" button on the top menu. A box should appear
  3. In the box type: natural=volcano
  4. Press 'build and run query'
  5. you should see marked objects representing volcanoes
  6. Congratulations!

You should see something like that, with circles marking points where volcanoes are marked.
