Listing project ideas for improving OpenStreetMap

Mapping projects are excluded from this list, only less direct projects are listed here.

Projects below are NOT reserved, feel free to work of them!

In general ideas are not worth much, 99.9% of input is actual work, not idea stuff.

Though pointless well done work is quite sad, so selecting a good target is worth spending some time on.

I created this list primarily for my own use, as I wanted to organise my chaotic TODO lists. Also, it will be useful when applying for grant supporting OpenStreetMap-related development. But it may be useful as inspiration for your own work, maybe GSoC proposal or something similar.

If you see mistakes, something unclear or it is unclear why working on a given project makes sense - let me know. It is intended to be human-readable in general.

In case that someone would be interested in sponsoring work on projects below - feel free to contact me. Note that in some cases I would first need to ask other involved people. For example in case of JOSM I am unsure what is the position of JOSM maintainers. It would be definitely fine for independent projects, where I am the main author and for StreetComplete. A/B Street

A/B Street is quite risky project. It is still too early to use it as a simulation for purposes of checking how traffic flow is influenced by potential changes. And it is unclear how much work is needed to reach that state, and project has a clear bus factor of 1.

Next danger is risk that goverments are not actually interested at all in real analysis and currently done simulations are purely smokescreens. And any actual simulation would be ignored.

But if it actually works and detailed OpenStreetMap data is actually used to simulate how infrastructure changes would affect traffic and such simulation would be mostly right and actually used to make decisions - it would be an amazing victory!

Also, I heard that Rust is great and I want to learn it.
