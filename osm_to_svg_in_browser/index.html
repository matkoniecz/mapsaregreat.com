Generation of vector maps from OpenStreetMap data.

  • general high zoom - area:highway included, general style - works well on scale of junction, neighbourhood, town, maybe also for entire cities
  • laser neighbourhood - depiction of a neighbourhood - generates map designed to be a template for laser-cut 3D tactile maps
  • laser road area - depiction of a single crossing - generates map designed to be a template for laser-cut 3D tactile maps
  • simple - very, very simple map style showing few elements. Useful if you are trying to make own and you want something as a base

Made using Lunar Assembler

See project page at Github. Feedback, contributions are welcomed!

Feel free to make pull requests and issues also about things such as typos and missing documentation.
